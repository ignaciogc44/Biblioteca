La finalidad de este proyecto es llevar el registro de los clientes, los libros y poder efectuar devoluciones y
préstamos de libros.
Permite consultar libros por su ID de ejemplar y clientes por su DNI
A su vez, al realizar un préstamo o una devolución, estos cambios se registran tanto en el perfil del cliente
como en el stock de libros, marcando el estado "Disponible" o "Prestado"
